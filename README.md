# CloudPaper
A live wallpaper for Android with an animation of the sky.

Built by Michael Chermside with assistance from the Claude AI.

# Privacy Policy

Last updated: December 12, 2025

CloudPaper is a live wallpaper application. The developer believes applications should not engage in any unnecessary tracking.

### DATA COLLECTION
CloudPaper does not collect any data whatsoever about its users.

### DATA USAGE
CloudPaper makes absolutely no network calls and thus uses no data.

### ADVERTISING AND TRACKING
CloudPaper does not use any analytics, advertising, or tracking services.

### DATA SHARING
CloudPaper does not share any data with third parties (also, it does not collect any data that COULD be shared).

### CONFIRM FOR YOURSELF
The source code for CloudPaper is available at https://github.com/mcherm/CloudPaper.

### CONTACT
If you have questions about this privacy policy, please contact the developer: Michael Chermside at mcherm@mcherm.com.
